# 🔗 SnapLink - URL Shortener

A modern, full-stack URL shortening service built with **Node.js**, **Express**, and **MongoDB**. This project was created to provide a clean and fast way to manage long, messy URLs.



## ✨ Features
- **Instant Shortening**: Converts long URLs into 6-character unique codes.
- **Modern UI**: A clean, "glassmorphism" styled interface.
- **Database Storage**: Uses MongoDB Atlas to store and persist link data.
- **Auto-Redirection**: Instantly redirects users from the short link to the original destination.

## 🛠️ Tech Stack
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (Mongoose ODM)
- **Frontend:** EJS (Template Engine), CSS3
- **Dev Tools:** Nodemon, Dotenv
